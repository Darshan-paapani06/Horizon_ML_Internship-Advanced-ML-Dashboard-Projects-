# AI Internship Projects Portfolio

This repository contains two advanced Machine Learning projects developed using Python, Streamlit, Scikit-Learn, Pandas, NumPy, and Matplotlib.

The projects are designed with modern real-time dashboards, interactive visual analytics, and enterprise-level UI architecture.

---

# Projects Included

## 1. AI Real Estate Analytics Platform

### Overview

An advanced real estate analytics platform that predicts property prices using Machine Learning algorithms and provides intelligent property recommendations based on user requirements.

### Features

- Real-time house price prediction
- Interactive live charts
- Dynamic market analytics
- Dark professional dashboard UI
- Location-based property recommendations
- Correlation heatmaps
- Property trend analysis
- User budget filtering
- Responsive analytics system

### Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

### Machine Learning Model

- Linear Regression

### Functional Modules

- Property Prediction Engine
- User Requirement Filtering
- Market Analytics Dashboard
- Dynamic Visualization System

## 2. AI Movie Recommendation Platform

### Overview

### Features

- Genre-based filtering
- Rating-based recommendations
- Real-time dashboard analytics
- Advanced visual reports
- Recommendation confidence analysis
- Interactive trend visualization
- Modern dark-themed UI

### Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

### Recommendation Technique

